# Java Student Utility App


## Description

A simple console-based Java application that perfoms multiple basic operations using a menu-driven system. this project is designed to demonstrate understanding of java Fundamentals such as variables, operators, loops, methods, decision making and user input.

## Features

- Add two numbers
- Check whether a number is EVEN or ODD
- Print numbers from 1 to N
- Display day of the week
- Exit application

## How to Run

1. Clone the repository
2. Open the Project in any Java IDE
3. Compile and run `StudentUtilityApp.java`
4. Follow the on-screen menu instructions
